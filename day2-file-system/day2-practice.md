## Commands used

CMD:
```bash
cd - change directory
dir - list files
mkdir - create folder
copy - copy file
del - delete file
```

PowerShell:
```bash
Set-Location - change directory
Get-ChildItem - list files
New-Item - create file/folder
Copy-Item - copy file
Remove-Item - delete file
```
