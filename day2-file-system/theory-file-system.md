# Day 2 - Windows File System

## What I learned
Today I learned how to navigate the Windows file system using CMD and PowerShell.

---

## Paths
Absolute path:
C:\Users\user

Relative path:
day2\

Special symbols:
.  - current directory
.. - parent directory

---

## Basic commands

CMD:
- cd - change directory
- dir - list files
- mkdir - create directory
- type nul > file name - create files
- del - delete file
- copy - copy file

PowerShell:
- Set-Location - change directory
- Get-ChildItem - list files
- New-Item - create items
- Remove-Item - delete items
- Copy-Item - copy file

---

## Summary
Understanding file navigation is essential for working in Windows and cybersecurity tasks.
